Vue d'ensembleOverview

Le driver Can contrôle directement le(s) contrôleur(s) CAN du MCU. Il gère l'initialisation des registres hardware, la configuration du baudrate (classic CAN 500 kbit/s ou CAN-FD jusqu'à 8 Mbit/s), les mailboxes TX/RX et les interruptions de transmission, réception et bus-off.The Can driver directly controls the MCU's CAN controller(s). It manages hardware register initialization, baudrate configuration (classic CAN 500 kbit/s or CAN-FD up to 8 Mbit/s), TX/RX mailboxes and transmission, reception and bus-off interrupts.

C — Can_Write & ISR 
#include "Can.h"

/* Can_Write called by CanIf */
Can_ReturnType result;
Can_PduType pdu;

pdu.id     = 0x123U;
pdu.length = 8U;
pdu.sdu    = txData;
pdu.swPduHandle = txPduHandle;

result = Can_Write(HOH_Tx_0, &pdu);
/* CAN_OK    → placed in mailbox
 * CAN_BUSY  → no mailbox free
 * CAN_NOT_OK → error */

/* CAN RX ISR (generated) */
void Can_0_RxIsr(void)
{
  /* 1. Read CAN ID, DLC, data
   *    from hardware mailbox
   * 2. Call CanIf_RxIndication(
   *      Hrh, CanId, DLC, data)
   * 3. Clear interrupt flag
   */
}

/* Bus-off ISR */
void Can_0_BusOffIsr(void)
{
  CanIf_ControllerBusOff(
    CAN_CONTROLLER_0);
  /* CanSM → ComM: NO_COM */
}

Bonnes pratiquesBest Practices

Conseil : Calculez les segments de temps (PROP, TSEG1, TSEG2, SJW) avec un outil de calcul CAN baudrate. Un sample point à ~80% est recommandé pour un réseau 500 kbit/s. Vérifiez la compatibilité CAN-FD avec tous les nœuds du réseau avant d'activer le data baudrate élevé.Tip: Calculate time segments (PROP, TSEG1, TSEG2, SJW) with a CAN baudrate calculator tool. A sample point at ~80% is recommended for a 500 kbit/s network. Verify CAN-FD compatibility with all network nodes before enabling high data baudrate.
