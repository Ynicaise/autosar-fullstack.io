Vue d'ensembleOverview

Le J1939Tp implémente les protocoles de transport définis par SAE J1939-21 pour les véhicules lourds (camions, bus, engins de chantier). Contrairement au CanTp (ISO 15765-2), il utilise des PGN (Parameter Group Number) spécifiques pour le contrôle de transport : TP.CM (60416) et TP.DT (60160).J1939Tp implements transport protocols defined by SAE J1939-21 for heavy-duty vehicles (trucks, buses, construction equipment). Unlike CanTp (ISO 15765-2), it uses specific PGN (Parameter Group Numbers) for transport control: TP.CM (60416) and TP.DT (60160).

C — J1939 BAM Sequence 
/* BAM TX sequence for PGN 65251
 * (Engine Configuration, 39 bytes):
 *
 * 1. J1939Tp sends TP.CM_BAM:
 *    PGN=0x00EC00 (TP.CM)
 *    Byte1: 0x20 (BAM control)
 *    Byte2-3: total size (39)
 *    Byte4: num packets (6)
 *    Byte5: 0xFF
 *    Byte6-8: PGN 65251
 *
 * 2. J1939Tp sends TP.DT packets:
 *    PGN=0x00EB00 (TP.DT)
 *    Packet 1: SN=1, 7 bytes data
 *    Packet 2: SN=2, 7 bytes data
 *    ...
 *    Packet 6: SN=6, 4 bytes + pad
 *
 * 3. Inter-packet delay: 50-200 ms
 *
 * No acknowledgment in BAM mode
 * (broadcast to all nodes).
 */

Bonnes pratiquesBest Practices

Conseil : Utilisez le mode CMDT pour les messages point-à-point critiques (avec acquittement). Le mode BAM est réservé aux broadcasts. Respectez les délais inter-paquets J1939 (50-200 ms) pour ne pas surcharger le bus.Tip: Use CMDT mode for critical point-to-point messages (with acknowledgment). BAM mode is reserved for broadcasts. Respect J1939 inter-packet delays (50-200 ms) to avoid bus overload.
