Vue d'ensembleOverview

L'EthIf fournit une interface unifiée vers un ou plusieurs contrôleurs Ethernet. Il gère le buffer management, le VLAN tagging, l'état du lien physique et l'interaction avec les modules TCP/IP stack (SoAd, TcpIp) et switch driver (EthSwt).EthIf provides a unified interface to one or more Ethernet controllers. It manages buffer management, VLAN tagging, physical link state and interaction with TCP/IP stack modules (SoAd, TcpIp) and switch driver (EthSwt).

C — EthIf Usage 
#include "EthIf.h"

/* TX: get buffer, fill, transmit */
void EthIfSendFrame(void)
{
  uint8* bufPtr;
  Eth_BufIdxType bufIdx;
  uint16 len = 64U;

  /* 1. Request TX buffer */
  EthIf_ProvideTxBuffer(
    EthIfConf_Ctrl_0,
    &bufIdx, &bufPtr, &len);

  /* 2. Fill payload */
  memcpy(bufPtr, payload, len);

  /* 3. Trigger transmission */
  EthIf_Transmit(
    EthIfConf_Ctrl_0,
    bufIdx, 0x0800, TRUE, len);
}

/* RX: EthIf_RxIndication()
 * dispatches to TcpIp / SoAd */
